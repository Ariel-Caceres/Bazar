# PREENTREGA REACT

Esta es un e-commerce que utiliza dummyjson como base de datos


## Funcionalidades
  - Filtrar y ver productos por categoría y marca.
  - Agregar y eliminar productos del carrito.
  - Ver detalles de los productos.

## Tecnologías usadas ⚙🚀
### Frontend 🎨✨
- React + Vite
- TypeScrips
- CSS
- ReactRouter
- FontAwesome
- useContext
### Backend 📁📊
- Express
- Cors
- Json file as Data Base
